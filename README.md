# LLM全流程实战培训课程

这是一个关于大型语言模型（LLM）从微调到部署的完整实战课程项目。

## 课程简介

**课程名称**: LLM全流程实战：从微调到部署  
**课程时长**: 2天（每天6小时）  
**课程目标**: 学员将掌握使用主流开源工具进行LLM微调、量化、部署的完整流程

### 技术栈
- **微调**: LLaMA Factory
- **量化**: llama.cpp
- **部署**: Ollama
- **API管理**: One-API
